# paint

<!-- badges: start -->

<!-- badges: end -->

An artisanally crafted set of print methods for `data.frame` family
rectangles.

## Installation

``` r
remotes::install_github("milesmcbain/paint")
```

## Example

``` r
library(paint)
#> Registered S3 method overwritten by 'paint':
#>   method           from
#>   print.data.frame base
```
